# Research, Reports and Publications

Public GitHub Pages site for Dr Dan van den Hoek's research outputs.

After GitHub Pages is enabled, the page will be available at the repository's GitHub Pages address.

## Data sources

The page loads journal articles and preprints from the TSI publications feed, and supplements these with local items stored in `research-extras.json`.

## Updating extra outputs

Edit `research-extras.json` for professional reports, university outputs, tools, research translation resources, datasets, or public technical documents.

Items with buttons such as `Open tool`, `Read article`, `Download`, or `Open report` must include a valid URL. Items without confirmed public links should use `Details pending`, `Link pending`, `Not publicly available`, or `Unavailable`.
